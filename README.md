# Curve Delivery Android

The Android app for delivery drivers on the Curve platform. Enables drivers to receive and manage delivery assignments, track order status, and navigate to pickup and drop-off locations.

## Tech Stack

- **Language:** Kotlin
- **Build System:** Gradle (Kotlin DSL)
- **Architecture:** MVVM with Dependency Injection (DI)

## Getting Started

Open in Android Studio and sync Gradle. Requires backend API endpoint configuration.
